# Contributing

Thanks for considering contributing to this portfolio template. A few guidelines to make contributions easy:

- Fork the repository and open a pull request with your changes.
- Keep changes small and focused (e.g., accessibility fixes, improved docs, additional templates).
- Update `README.md` when changing usage or templates.
- Add tests or preview screenshots where helpful.

Customizing the template locally:

1. Replace placeholders in `site/index.html` (examples: `{{YOUR_NAME}}`, `{{SHORT_BIO}}`, `{{EMAIL}}`, `{{SITE_URL}}`).
2. Replace `{{OG_IMAGE_URL}}` with a hosted image URL for social previews.
3. Edit or replace example project cards in the `#projects` section.
4. The template uses Tailwind via CDN; for production you can integrate a local Tailwind build.

If your change touches design, prefer to keep it responsive and accessible (semantic HTML, focus management, ARIA where appropriate).
